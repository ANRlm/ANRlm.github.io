KeyMap

Vim

Vim 插入键位置和移动光标键说明

O 上一行首
o 下一行首
I 当前行首
i 光标前
A 当前行尾
a 光标后
^ 当前行首
$ 当前行尾
yy 复制内容
p 粘贴内容
dd 删除内容
G 最后一行
gg 第一行
数字+G 跳转对应行
u 撤销
ctrl+f (forward) 向前翻页
ctrl+b (backward) 向后翻页
ctrl+u (up) 向上翻半页
ctrl+d (down) 向下翻半页

查找

/ 向下查找
？ 向上查找
n (next)下一结果
N 上一结果
\c 不区分大小写

替换

在vim尾行模式

(如替换当前行则省略n1,n2)(/g为全局变量，如不加则进仅替换每一行第一个匹配到的内容)

joshto

cw = change word 重命名

yy 复制

pp 粘贴

zh 显示/隐藏文件

aerospace

option + shift + ; 服务模式

option + / 水平和垂直排列切换

option + , 手风琴排列模式(按下option+/返回平铺模式)

fzf

control+t 开启 fzf

\ + tab 开启 fzf; 再次按下 tab 进行多选, shift+tab 取消勾选

control + r 命令历史记录

bat(better cat)

bat + 文件路径 预览文件内容

tldr(better man)

tlrc + 命令 命令用户手册

zoxide

z + 文件夹名称 打开对应文件夹(无需输入完整路径)

z + 空格 + tab 给出当前路径下所有选项
